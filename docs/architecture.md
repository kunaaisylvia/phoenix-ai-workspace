# Phoenix AI Workspace Vision

Phoenix AI Workspace is an AI-powered engineering workspace built for software developers.

Its mission is to help developers:

- Store engineering knowledge
- Review and generate code
- Analyze datasets using AI
- Build a long-term knowledge vault
